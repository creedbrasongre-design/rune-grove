# Rune Grove Quest — No Front Image Version

This version removes the front/preview image entirely.

The playable game now uses only the drawn RPG map inside the canvas. The finished concept-art scene image is not used as the page preview or the canvas background.

## Controls

- Move: Arrow keys or WASD
- Interact: E or Space
- Walk through tall grass for battles
- Reach Level 3 and enter the portal
